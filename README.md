# LePort

LePort converts robot datasets to LeRobot Dataset v3.

| Source format | Supported |
|---|:---:|
| robomimic HDF5 | ✅ |
| ALOHA HDF5 | ✅ |
| ManiSkill HDF5 + JSON | ✅ |
| Zarr | |
| RLDS | |
| ROS bag | |
| MCAP | |
